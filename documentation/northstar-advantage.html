NorthStar Advantage Computer

THIS DOCUMENT IS A WORK-IN-PROGRESS

The NorthStar Advantage, shortened to Advantage throughout this page, is a z80-based business focussed all-in-one microcomputer designed and sold by NorthStar Computer Systems, Inc. cerca 1981.

Computer Specs

CPU z80A 4MHz
Memory 64KB
BOOT ROM 2KB
Video Memory 20KB
Display 640x240px 4:3 11" Green Monochrome CRT
Floppy Drives Dual [model], 10 Hard-Sectors, Supports both DSDD(360KB) and SSDD(170KB)
Hard Disk Optional MFM Hard Disks: 5MB, 10MB, 15MB
Keyboard 91 key, N-key rollover (w/ 7-key buffer)
Extra Specs Speaker, Parallel Port, Syncronous/Asyncronous Serial, optional 8088 co-processor

Unofficial Documentation

Floppy Sector Size

The Technical Manual lists the Sector size as 256B on page [PAGE_A], and then again as 128B on page [PAGE_B]. The first entry of 256B is correct; the later 128B specification is a miss-print.

Floppy CRC Calculation

The Technical Manual provides incorrect formula for CRC calculation.

While disassembling CP/M's disk i/o BIOS procedures, I noticed that the CRC calculation did not match what was documented. Writing NSAE proved that point further.

Do note that the Advantage both calculates and checks CRC in software only, the core calc/check logic is handled by the BIOS and/or Operating System. This correction is known to hold true for NorthStar's CP/M and NSGDOS OSs'.
