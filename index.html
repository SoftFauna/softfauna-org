sfsuppliment

SoftFauna Suppliment is a C library which acts as a suppliment ontop of the C standard library. It provides many usefull macro's such as ARRLEN, MIN, MAX, and ABS, aswell as extending the available string and array operations. SFSuppliment adds many highlevel convience features to ANSI C to speed up developement times and overhead.

hemlock-argparser

SoftFauna's Arguement Parser takes the pain out of parsing console arguements, flags, and options. Implementing an automated parser, type checking, type conversions, returns, and error logging all through one API. Additionally, generate brilliant help and version pages at runtime using hemlock-argparser's generate commands and optional coloured printing.
